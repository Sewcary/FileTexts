This README file contains important information about SimpleMind Pro Trial

TRIAL RESTRICTIONS

The SimpleMind Pro Trial is for evaluation and trial purpose only.
The SimpleMind Pro Trial has the same functionality as SimpleMind Pro, with the
only restriction that it will expire 30 days after first run.

------------------------------------------------------------------------------
  MINIMUM SYSTEM REQUIREMENTS
------------------------------------------------------------------------------

SimpleMind requires Windows 7 or newer.
SimpleMind is designed to work on a 1024x800 monitor minimum.

-----------------------------------------------------------------------------
  INSTALLATION
------------------------------------------------------------------------------

You must have administrator rights when installing SimpleMind.

=*= Shut down SimpleMind before installing
=*= Run the Setup
=*= (Re-)Start SimpleMind desktop Edition
